# 애니 추천 프로그램

장르를 선택하면 선택한 장르와 가장 잘 맞는 애니를 추천해주는 파이썬 GUI 프로그램입니다.

## 파일 구성

- `main.py`: 프로그램 실행 파일
- `ui.py`: tkinter 화면 구성
- `recommender.py`: 추천 점수 계산 로직
- `scoring.py`: 1순위 장르와 추가 장르 가중치 계산
- `data.py`: 애니 데이터, 표시 장르, 장르별 가까운 정도

## 실행 방법

```bash
python3 main.py
```

창이 열리면 1순위 장르를 먼저 고르고, 추가로 반영할 장르를 선택한 뒤 `추천받기` 버튼을 누르면 됩니다.

## 데이터 형식

각 애니 데이터의 `genre_affinity` 값은 해당 애니가 장르에 얼마나 가까운지 나타냅니다.
값이 높을수록 추천 점수에 더 크게 반영됩니다.
