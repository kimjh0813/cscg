# 애니 추천 프로그램

장르를 선택하면 선택한 장르와 가장 잘 맞는 애니를 추천해주는 파이썬 GUI 프로그램입니다.
Kivy를 사용해 브라우저가 아닌 데스크톱 앱 창으로 실행됩니다.

## 파일 구성

- `main.py`: 프로그램 실행 파일
- `ui.py`: Kivy 화면 구성
- `theme.py`: 앱 색상 테마 값
- `recommender.py`: 추천 점수 계산 로직
- `scoring.py`: 1순위 장르와 추가 장르 가중치 계산
- `data.py`: 애니 데이터, 표시 장르, 장르별 가까운 정도
- `assets/fonts/`: 프로젝트에서 사용할 한글 폰트 위치
- `requirements.txt`: 실행에 필요한 파이썬 라이브러리

## 실행 방법

```bash
python3.11 -m pip install -r requirements.txt
python3.11 main.py
```

실행하면 데스크톱 앱 창이 열립니다. 1순위 장르를 먼저 고르고 추가로 반영할 장르를 선택한 뒤 `추천받기` 버튼을 누르면 됩니다.

## 폰트 설정

다른 컴퓨터에서도 한글이 깨지지 않도록 `assets/fonts/NotoSansKR-Regular.ttf`를 프로젝트에 포함했습니다.
프로그램은 운영체제 시스템 폰트를 사용하지 않고 이 프로젝트 폰트 파일만 사용합니다.

## 데이터 형식

각 애니 데이터의 `genre_affinity` 값은 해당 애니가 장르에 얼마나 가까운지 나타냅니다.
값이 높을수록 추천 점수에 더 크게 반영됩니다.
